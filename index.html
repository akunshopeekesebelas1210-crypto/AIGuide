Implikasi untuk Landing Page

Hook dalam 3 detik pertama. Visual lebih kuat dari teks panjang. Chunk kecil-kecil. Satu CTA yang jelas. Authentic beats polished.

Formula Konten Anti-Bosan Gen Z

  • Hook visual atau bold statement di atas the fold
  • Chunk pendek — maks 2–3 kalimat per blok
  • Gunakan angka dan visual break
  • Progressive disclosure — kasih pilihan lanjut atau skip
  • Authentic tone — ngobrol, bukan presentasi
  • Satu CTA utama yang mencolok

3 Workflow

AI
BUILD
FLOW

4 tahap dari ide ke landing page yang jalan di laptop lo. Sistematis, dan tetap lo yang control creative-nya.

Prinsip Utama

AI itu asisten, bukan pengganti ide lo. Lo yang kasih arah, AI yang eksekusi detail. Kreativitas tetap 100% punya lo.

Stack yang Digunakan

Ide Lo Sendiri

Nentuin bisnis, brand name, produk, target market, dan "feeling" yang lo mau.

Creative Direction

AI / Web Research

Cari referensi visual, kompetitor, dan konten. Pinterest, Dribbble, Google.

Research & References

Gemini Pro (Reasoning Mode)

Kasih brief mentah lo, minta Gemini expand jadi prompt detail untuk Claude.

Prompt Architect

Claude Sonnet 4.6 Extended

Paste prompt dari Gemini. Build HTML/CSS/JS lengkap — UI, UX, fungsi, animasi, form order.

Build UI/UX + Logic

Run di Laptop

Save file .html, buka di browser. Upload ke Netlify / GitHub Pages untuk live URL gratis.

Local Development

Detail Per Tahap

Cari 5–10 contoh thrift store website yang lo suka. Pinterest untuk moodboard warna dan layout. Dribbble untuk UI inspiration. Screenshot semua, simpan di folder.
Buka gemini.google.com, aktifkan mode Deep Research atau 2.0 Flash Thinking. Ketik kasual — ceritakan bisnis lo, referensi visual, dan fungsi yang dibutuhkan. Minta Gemini generate "detailed UI/UX prompt for Claude AI."
Buka claude.ai, pilih model Sonnet 4.6, aktifkan Extended Thinking. Paste prompt dari Gemini. Iterate jika perlu — minta revisi per section langsung di chat yang sama.
Copy code dari Claude, simpan sebagai index.html. Buka di Chrome, inspect dengan DevTools untuk cek mobile view. Kalau oke, drag & drop ke Netlify.com untuk live URL gratis.

Sesimpel ini cara pakainya

Gemini sudah generate prompt yang panjang dan detail. Copy semua teksnya, buka claude.ai, pilih Claude Sonnet 4.6, aktifkan Extended Thinking, lalu paste.

  • Buka claude.ai → New Chat
  • Pilih model: Claude Sonnet 4.6
  • Aktifkan Extended Thinking (toggle di atas input)
  • Paste seluruh prompt dari Gemini
  • Tunggu Claude build — biasanya 1–2 menit
  • Copy hasilnya → simpan sebagai index.html

Kalau hasilnya belum pas?

Langsung bilang ke Claude di chat yang sama. Contoh: "perbesar font namanya", "ganti warna tombol jadi hitam", "tambahkan countdown timer". Claude ingat konteks.

3 Deploy Gratis

Live dalam 3 menit:

Deploy ke Netlify (tanpa coding) 
1. Simpan code dari Claude
   → File → Save As → index.html
   → Taruh di folder: /deadstock-id/

2. Buka netlify.com
   → Login gratis (bisa pakai GitHub)
   → Klik "Add new site"
   → "Deploy manually"

3. Drag & drop folder
   → Drag folder /deadstock-id/ ke browser
   → Tunggu 30 detik

4. URL lo jadi:
   → deadstock-id.netlify.app ✓
   → Bisa custom domain kapanpun

// GitHub Pages juga bisa — butuh basic git
