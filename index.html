💡 Implikasi untuk Landing Page

Hook dalam 3 detik pertama. Visual > teks panjang. Chunk kecil-kecil. Satu CTA yang jelas. Authentic beats polished.

Formula Konten Anti-Bosan Gen Z

  • Hook visual atau bold statement di atas the fold
  • Chunk pendek — maks 2–3 kalimat per blok
  • Gunakan angka, emoji, dan visual break
  • Progressive disclosure — kasih pilihan untuk lanjut atau skip
  • Authentic tone — ngobrol, bukan presentasi
  • Satu CTA utama yang mencolok
1 Industri

BISNIS
PERTAMA
GEN Z

Berdasarkan data dan tren 2025–2026, ini satu industri yang paling konkret, low-cost, dan native Gen Z.

🧥 Online Thrift & Vintage Store

Paling banyak dipilih Gen Z sebagai bisnis pertama. Mulai dari Depop, Instagram, sampai website sendiri. Modal awal kecil, kurasi produk bisa pakai taste sendiri, dan market-nya nggak ada habisnya.

Low Capital Social-Native Sustainable Kurasi = Brand
F&B memang populer (cafes, plant-based), tapi modal awalnya lebih besar dan ada risiko fisik. Thrift store bisa 100% online dari hari pertama, cocok untuk testing sebelum invest lebih besar.
Content creation butuh audience dulu, baru monetize. Thrift store bisa langsung jual dari hari pertama. Tapi keduanya bisa dikombinasi — banyak thrift store sukses karena konten TikTok-nya.

Yang dibutuhkan bisnis ini:

  • Landing page mobile-first (mayoritas pembeli Gen Z buka HP)
  • Sistem pemesanan yang simple (form WhatsApp / direct order)
  • Galeri produk yang clean dan aesthetic
  • Brand identity yang kuat — nama, warna, font

Local Development

Detail Per Tahap

Cari 5–10 contoh thrift store website yang lo suka. Pinterest untuk moodboard warna & layout. Dribbble untuk UI inspiration. Screenshot semua, simpan di folder. Ini bahan utama lo kasih ke Gemini.
Buka gemini.google.com, aktifkan mode penalaran/pro. Ceritakan bisnis lo secara lengkap: nama brand, produk, target market, referensi visual yang lo suka, fungsi apa yang dibutuhkan (galeri, form order, dll). Minta Gemini generate "detailed UI/UX prompt for Claude AI."
Buka claude.ai, pilih model Sonnet 4.6 dengan fitur Extended Thinking aktif. Paste prompt dari Gemini. Tambahkan: "Build as single HTML file, mobile-first, with embedded CSS and JS. Include order form that sends to WhatsApp." Iterate jika perlu — minta revisi per section.
Copy code dari Claude, simpan sebagai index.html. Buka di Chrome/Safari di laptop. Inspect dengan DevTools (F12) untuk cek mobile view. Kalau udah oke, drag & drop ke Netlify.com atau push ke GitHub Pages untuk live URL gratis.

2 Prompt → Claude

Contoh hasil dari Gemini → Paste ke Claude:

🤖 Prompt Claude (dari Gemini, contoh) 
Build a complete single-file HTML landing page 
for DEADSTOCK.ID, a Gen Z vintage thrift store.

Design System:
- Background: #FAFAF7 (off-white)
- Text: #0A0A0A (near black)
- Accent: #E8D84B (yellow)
- Font: Import Bebas Neue (display) + 
  DM Sans (body) from Google Fonts

Sections (in order):
1. HERO — Full viewport, brand name HUGE in 
   Bebas Neue, subline italic, CTA button
   
2. PRODUCT GRID — 6 product cards with:
   - Placeholder image (gray box)
   - Item name, size, price
   - "ORDER" button per card
   
3. HOW TO ORDER — 3 steps numbered, clean layout

4. DROP SCHEDULE — Next drop countdown timer

5. FOOTER — Brand name + WhatsApp button

Functionality:
- All ORDER buttons open WhatsApp link:
  wa.me/6281234567890 with auto-message
- Smooth scroll between sections
- Mobile-first CSS (max-width 430px optimal)
- Subtle scroll animations (IntersectionObserver)

Output: Single .html file, all CSS+JS embedded.

💡 Tips Iterasi di Claude

Setelah Claude generate, kalau ada yang kurang, langsung bilang: "Perbesar font heronya 20%. Ganti warna tombol jadi hitam. Tambahkan section Instagram feed placeholder." Claude ingat konteks sebelumnya.

3 Deploy Gratis

Live dalam 3 menit:

🚀 Deploy ke Netlify (tanpa coding) 
1. Simpan code dari Claude
   → File → Save As → index.html
   → Taruh di folder: /deadstock-id/

2. Buka netlify.com
   → Login gratis (bisa pakai GitHub)
   → Klik "Add new site"
   → "Deploy manually"

3. Drag & drop folder
   → Drag folder /deadstock-id/ ke browser
   → Tunggu 30 detik

4. URL lo jadi:
   → deadstock-id.netlify.app ✓
   → Bisa custom domain kapanpun
